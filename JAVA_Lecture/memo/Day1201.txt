String 클래스 메소드

compareTo => int 타입을 리턴함 (1,0,-1)
-> 정렬하는 메소드임 (오름차순, 내림차순)
-> 두개를 비교했을때 (한글도 비교가능함)

replace 
-> 문자 대체하는 메소드

substring
->인덱스를 해서 문자를 가져오고싶은 것만 가져올수있음

String.valueOf(모든 타입이올수있음 )
-> 엄청 유용함
static String 이 리턴됨. 즉, 매개변수 값을 스트링으로 변환해줌
