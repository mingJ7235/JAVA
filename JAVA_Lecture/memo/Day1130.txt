Object -clone ()
얕은 복제 - Object클래스 - clone() 사용시
	기본형 값 복사
	객체형 주소 복사
	(원본과 카피가 같은 주소를 가르킴)
	
	Car c = new Car ();
	
얕은 복제를 하면 원본을 바꾸면 카피가 바뀌고,
참조변수의 가르키는 주소가 동일하므로
참조변수의 값을 바꾸면값이 같이 바뀐다.

깊은 복제 
- 자손클래스가 부모 클래스 Object클래스 - clone() 사용
- 자손클래스가 재정의하여 clone() 한번 더 수행한다.

	변수의 가르키는 주소를 다른 주소로 준다.
	참조변수의 가르키는 주소를 다른 것으로 복사하므로
	원본의 참조변수 값을 바꿔도 깊은 복제의 참조변수 값은 바뀌지 않는다.
