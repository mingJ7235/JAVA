JAR 파일 만드는 방법
-> 이클립스가 없어도 사용할 수 있게해주는 것.

Export -> runnable JAR 


-----------------
dir : 디렉토리 (폴더) 목록보기
cd : change directory 디렉토리 변경
cd .. : 앞의 디렉토리로 가기
cd 폴더명 : 해당폴더로 이동하기

jar 파일을 실행하는 명령
-> java -jar 파일명.jar

예외처리하기

프로그램오류
- 컴파일 에러 : 컴파일 시 발생하는 에러 (문법적으로 틀린것)
	=> IDE가 코딩시 알려준다
- 런타임 에러 : 실행시 (빌드단계시) 발생하는 에러
	=> run을 돌릴때 에러가 난다. 
- 논리적 에러 : 실행은 되지만 의도와 다르게 동작하는 경우
	=> 답이 이상하게 나올때. 가장 잡기 어려운 에러

에러 : 프로그램 코드에 의해서 수습될 수 없는 심각한 오류
	=> 심각함
예외 : 프로그램 코드에 의해서 수습될 수 있는 다소 미약한 오류
	=> 강도가 낮음 문제임.
